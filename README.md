# Disk-Cleanup
A technician-friendly PowerShell disk cleanup tool via GUI for Windows 10/11. Includes Safe, Aggressive, and Ultra profiles. Scans temp folders, Downloads, browser caches, system/app logs, and Windows Update data. Sortable grid, filters, selection tools, and detailed action log.

CMD+PS1 and EXE are the same app, just different execution. Use as see fit. 

> With PS1 file, drop the PS1 and CMD in a location, and double click the CMD file **RunMe-v2.cmd** and the app will launch.
